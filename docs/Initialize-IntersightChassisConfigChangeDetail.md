For example, this could happen when the user creates a server profile and has not deployed the profile yet.\n* `Modified` - The &apos;update&apos; operation/state.Indicates some of the desired configuration changes specified in a profile have not been been applied to the associated device.This happens when the user has made changes to a profile and has not deployed the changes yet, or when the workflow to applythe configuration changes has not completed successfully.\n* `Deleted` - The &apos;delete&apos; operation/state.Indicates a configuration profile has been been disassociated from a device and the user has not undeployed these changes yet.
