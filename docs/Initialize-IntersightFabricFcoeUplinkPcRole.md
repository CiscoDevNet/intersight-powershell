Supported speeds are 25Gbps and 100 Gbps.\n* `Cl74` - Forward error correction option &apos;cl74&apos;. Supported speeds are 25Gbps.\n* `rs-cons16` - Forward error correction option \&quot;rs-cons16\&quot;.
